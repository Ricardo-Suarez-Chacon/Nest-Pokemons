# Ejectar en desarrollo

1. clonar el repositorio
2. ejectar

```
yarn install
```

3. Tener Nest Cli instalado

```
npm i -g @nest/cli
```

4. levantar la base de datos

```
docker-compose up -d
```

## Stack usado

- Mongo db
- Nest
